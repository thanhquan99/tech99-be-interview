# tech99-be-interview

## Overview

### Problem 4: Three Ways to Sum to N
Provide 3 unique implementations of a function `sum_to_n(n)` in TypeScript that returns the summation from 1 to `n`. Each implementation includes commentary on its complexity and efficiency.

### Problem 5: A Crude Server
Build a backend server using ExpressJS and TypeScript with full CRUD functionality. The service connects to a database for persistence and exposes endpoints to create, list, get, update, and delete resources.

### Problem 6: Architecture
Design a specification for a live-updating scoreboard module. Includes documentation, an execution flow diagram, and considerations for preventing unauthorized score manipulation.
